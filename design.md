# 교통사고 토탈케어 앱 — 디자인 설계 문서

- **앱 이름:** 차케어 (CareCar)
- **슬로건:** 사고 났을 때, 당황하지 마세요
- **색상 팔레트:**
  - Primary (네이비): #1A2B4C — 신뢰, 전문성
  - Accent (블루): #3182CE — 행동 유도, 링크
  - Warning (오렌지): #DD6B20 — SOS/긴급, 강조
  - Background (흰색): #FFFFFF

## 화면 목록 (Screen List)

### 1. 홈 (Home) — `app/(tabs)/index.tsx`
- **상단 헤더:** 앱 로고 + "안녕하세요, 운전자님" 인사말 + 알림 아이콘
- **내 차량 카드:** 차량 이름, 예상 시세, 마지막 업데이트 시간
- **SOS 긴급 접수 버튼:** 화면 중앙 대형 빨간/오렌지 버튼 (슬라이드 잠금 해제 UX)
- **오늘의 꿀팁 카드:** 데일리 교통사고 정보 1건 미리보기
- **위험 구간 알림 배너:** 현재 위치 기반 위험 구간 여부 표시

### 2. 사고 처리 (Care) — `app/(tabs)/care.tsx`
- **진행 중인 사고 없을 때:** AI 파손 분석 테스트 CTA + 내 주변 파트너 바로가기
- **진행 중인 사고 있을 때:** 타임라인 대시보드 (수리중/렌트이용중/합의대기 등 단계 표시)
- **파트너 매칭 섹션:** 공업사/렌터카/병원/변호사 카테고리별 추천 카드

### 3. 커뮤니티 (Feed) — `app/(tabs)/feed.tsx`
- **블랙박스 피드:** 사고 영상 카드 + AI 과실 비율 뱃지 (예: 가해자 70% / 피해자 30%)
- **데일리 꿀팁 피드:** 카드 뉴스 형태의 정보 콘텐츠 목록

### 4. 내 주변 (Map) — `app/(tabs)/map.tsx`
- **지도 뷰:** expo-maps 기반 지도 + 제휴 파트너 마커
- **파트너 필터 탭:** 공업사 / 렌터카 / 병원 / 변호사
- **하단 시트:** 선택된 파트너 상세 정보 (이름, 평점, 거리, 전화)

### 5. 마이페이지 (My) — `app/(tabs)/my.tsx`
- **차량 관리:** 등록 차량 목록, 보험증권 OCR 등록
- **사고 이력:** 과거 사고 처리 내역 타임라인
- **설정:** 알림 설정, 위험 구간 알림 ON/OFF

## 핵심 사용자 플로우

### 사고 발생 플로우
홈 SOS 버튼 → 슬라이드 잠금 해제 → 위치 자동 감지 → 사고 유형 선택 → 사진 촬영 가이드 → AI 파손 분석 → 파트너 매칭 추천 → 사고 처리 타임라인 시작

### 평상시 DAU 플로우
홈 진입 → 내 차 시세 확인 → 오늘의 꿀팁 읽기 → 블랙박스 피드 구경 → 위험 구간 알림 수신
